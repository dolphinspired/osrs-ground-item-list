What are these lists for?

These lists can be used to filter out or highlight item names which appear over items on the ground in Old School Runescape. I made this little webpage because editing and copying large text lists like this is very difficult on mobile, and nearly impossible from within the mobile OSRS client.

How were the items in these lists chosen?

These are the just the lists I currently use on my Ultimate Ironman account. If you just need some quick lists to clear up the clutter on your screen, these are a pretty good place to start.
You can also edit these lists before copying them and save any changes you make to your browser's cache. Your custom list will be restored when you refresh or re-open this page.

Which game clients can I use this with?

You can use these lists with any OSRS game client which supports ground item labels, which currently includes:

  • RuneLite
  • Old School RuneScape on Steam
  • Old School RuneScape on Android
  • Old School RuneScape on iPhone and iPad
Ground items are not yet available in the official OSRS client that launches from the Jagex Launcher. However, you can install and launch RuneLite from within the Jagex Launcher instead.

How to use with Official Clients

Open "All Settings" from within the game window. Search for the "Popout panel" setting, and make sure it's enabled.
Click the backpack icon on the popout panel to the right, and select "Ground Items" from the dropdown at the top of this panel.
If you want to keep your existing list, add a comma to the end of your existing list, then paste the list from this webpage at the end.
If you want to overwrite your existing list, simply delete what's already in this box and paste the new list in.

How to use with RuneLite

Click the wrench icon on the right panel and search for the "Ground Items" plugin.
Click the gear icon to open the settings for this plugin, and paste your lists into the appropriate fields.
If you want to keep your existing list, add a comma to the end of your existing list, then paste the list from this webpage at the end.
If you want to overwrite your existing list, simply delete what's already in this box and paste the new list in.
